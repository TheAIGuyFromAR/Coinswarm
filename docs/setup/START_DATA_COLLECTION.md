# Start Data Collection

This document provides instructions for starting the data collection process in Coinswarm.

## 1. Prerequisites
- Data provider API keys
- Configured `.env` file

## 2. Steps
1. Run `fetch_historical_data.py` for backfill
2. Run `fetch_multi_source_data.py` for live updates

## 3. Monitoring
- Check logs for errors
- Use monitoring dashboard for status
