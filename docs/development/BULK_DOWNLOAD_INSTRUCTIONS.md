# Bulk Download Instructions

This document provides instructions for bulk downloading historical data for Coinswarm backtesting and analysis.

## 1. Prerequisites
- Python 3.8+
- `requests`, `pandas` libraries

## 2. Usage
1. Run `bulk_download_historical.py`
2. Specify product and date range
3. Data will be saved to `data/` folder

## 3. Notes
- API rate limits may apply
- See `API_RATE_LIMITS.md`
